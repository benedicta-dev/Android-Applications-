# 🎓 Grade Calculator (Dart)

## 📱 Project Overview

The **Grade Calculator** is a simple mobile application developed using **Dart and Flutter**.
The app allows students to enter their scores and automatically calculates their **average score and final grade**.

This project was created as part of a **Continuous Assessment (CA)** to demonstrate basic **mobile app development using Flutter**.

 ✨ Features

* Input multiple subject or test scores
* Automatic average calculation
* Grade determination based on score
* Clean and simple user interface
* Real-time result display

---

 🛠️ Technologies Used

* **Dart**
* **Flutter**
* **Android Studio / VS Code**
* **Flutter SDK**
